# AppSheet Bots

## Overview

This directory documents the automation workflows used by the application.

AppSheet Bots are responsible for executing actions automatically whenever predefined events occur.

Current automations include:

- Expedition completion detection.
- PDF generation.
- Label creation.
- File storage.

Each Bot will be documented individually, including:

- Trigger
- Event
- Condition
- Process
- Tasks
- Dependencies
- Expected Result

The goal of this documentation is to simplify maintenance and provide a clear understanding of the application's automation architecture.
