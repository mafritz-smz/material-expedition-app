# Virtual Columns

## Overview

Virtual Columns calculate information dynamically.

Unlike physical columns, they do not store data permanently.

Instead, values are calculated every time the application loads the corresponding record.

## Typical Uses

- Current scanned quantity.
- Remaining quantity.
- Total scanned weight.
- Expedition progress.
- Related records.

## Benefits

Virtual Columns eliminate redundant information while keeping calculations always up to date.
