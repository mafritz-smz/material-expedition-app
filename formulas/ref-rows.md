# REF_ROWS()

## Purpose

Creates parent-child relationships between tables.

## Usage

`REF_ROWS()` allows a Production Order to display all associated scan records automatically.

This relationship is fundamental to the application's navigation and data organization.

## Benefits

- Simplifies navigation.
- Improves readability.
- Eliminates manual filtering.
- Keeps related records synchronized automatically.
