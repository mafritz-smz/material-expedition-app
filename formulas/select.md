# SELECT()

## Purpose

Returns one or more records that satisfy a given condition.

## Usage

`SELECT()` is used whenever the application needs to retrieve data based on business rules.

Typical use cases include:

- Calculating total scanned weight.
- Counting scanned pieces.
- Finding related records.
- Filtering information by Production Order.

## Why It Is Used

`SELECT()` allows the application to calculate dynamic values without storing redundant information.

This keeps the data model simple and flexible.
