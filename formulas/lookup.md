# LOOKUP()

## Purpose

Retrieves a single value from another table.

## Usage

`LOOKUP()` is primarily used to obtain information from the material database after a barcode is scanned.

Typical examples include:

- Material description
- Weight
- Machine
- Lot
- Color

## Why It Is Used

Using `LOOKUP()` avoids duplicated data entry and guarantees that information always comes from a single source.

This improves consistency throughout the application.
